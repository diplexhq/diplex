## Dependency Resolution (Resolver)

The resolver builds two indexes from `ParsedData` and performs BFS traversal from facade interfaces:

1. **`typeIndex`** — maps normalized provider Result types to candidates. Generic types are normalized: all parameters `T`, `T0`, `T1` → `T`.
2. **`interfaceMethodIndex`** — maps method signatures to providers implementing them. Non-generic methods: `"Name(params)(results)"`, generic: bare method name.

**Resolution process:**
- `resolveFacades()` — filters DI facade interfaces by `-di` directories
- `resolveProviders()` — BFS traversal from facade method return types, collecting all transitive dependencies
- `findProviderCollection()` — lookup by type (typeIndex) or by interface (interfaceMethodIndex)
- Generic constraints narrowed via `compareType()`, Cartesian product for multi-constraint

**Result:** `ResolvedData{Facades: Interfaces, Providers: map[Parameter]ProviderCollection}`

## Key Types

### SourceFiles channel

```go
// domain.SourceFiles — channel of source file paths
type SourceFiles chan SourceFile
```

Scanner produces a closed channel via goroutine. Parser consumes it with 4 workers.

### domain.ParsedData

```go
// domain.ParsedData - parsed dependency graph from source files
type ParsedData struct {
    Providers  Providers       // New* constructor functions
    Interfaces Interfaces      // interface methods + struct methods (RealType: true)
}

// domain.Provider - constructor signature with generic support
type Provider struct {
    Pkg       string            // package path
    Name      string            // function name (New*)
    Arguments []Parameter       // parameter type strings
    Result    Parameter         // return type string
    Generic   map[string][]string // T → [constraint types]
    Error     bool              // returns error
}
// Provider.Id() returns synthetic ID: pkg + "." + name

// domain.MethodMap - method name to contract
type MethodMap map[FunctionName]MethodContract

// domain.MethodContract - a single method signature
type MethodContract struct {
    Arguments []Parameter
    Results   []Parameter
}
```

## How to Run

```bash
# Run from target project root
go tool diplex

# Custom scan dirs
go tool diplex -scan internal,pkg

# Custom output
go tool diplex -out custom/dir

# Skip go.mod, set module manually
go tool diplex -module example.com/my/project

# Custom skip pattern
go tool diplex -skip "(generated|mocks|_test\.go)"

# Verbose / Silent
go tool diplex -v
go tool diplex -s
```

## Generated Files

`internal/generated/diplex/`
- `di.go` — DI facade (auto-generated, implements `di.DI` interface)

The DI facade is generated per directory listed in `-di` flag. Each DI directory produces one `.go` file with the `DI` struct and its methods that resolve all dependencies using `sync.OnceValue`.

### Usage Example

```go
// Your source code (scanned by diplex):

// 1. Interface
type UserService interface {
    GetUser(id string) (*User, error)
}

// 2. Implementation
type service struct { repo *UserRepo }
func (s *service) GetUser(id string) (*User, error) { /* ... */ }

// 3. Provider
func NewService(repo *UserRepo) (*service, error) {
    return &service{repo: repo}, nil
}

// After running diplex, use the generated container:

func main() {
    // 1. Create container (generated by diplex)
    c := diplex.NewDI()

    // 2. Access services — dependencies resolved automatically
    svc := c.UserService()

    // 3. Use
    user, err := svc.GetUser("user-123")
}
```

## Recognition Patterns

**Provider:**
```go
func NewX(deps...) (*X, error?)  // ✓
func NewRepo[T A | B]() *Repo[T] // ✓ generic → concrete instantiation
```

**Interface:**
```go
type Interface interface {       // ✓
    Method(args) results
}
type Repository[T any] interface { // ✓ generic interface
    Get(id int) (T, error)
}
```

**Implementation:**
```go
func (x *X) Method(args) results // ✓
func (r *Repo[Order]) Get(id int) (Order, error) // ✓ generic receiver
```

**Slice Collection:**
```go
func NewServer(handlers []Handler) *Server // ✓ aggregates all Handler providers
```
